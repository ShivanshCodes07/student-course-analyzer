# 📊 Flask Web App – Student & Course Marks Analyzer

This is a Flask-based web application that allows users to analyze student and course performance from a CSV file. Users can:

- 🔎 Search by **Student ID**: view all enrolled courses and total marks
- 📊 Search by **Course ID**: view average, highest marks, and histogram of scores
- 🚫 Receive error messages if the ID is invalid

---

## 🛠️ Technologies Used

- Python 3
- Flask
- Pandas
- Matplotlib
- HTML5 + CSS
